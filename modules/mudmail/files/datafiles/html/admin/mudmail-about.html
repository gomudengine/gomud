Mudmail

Module overview and room tag reference.

Description

The Mudmail module provides an in-game mailbox system. Players can send messages to other players, including gold and item attachments. Messages are delivered to the recipient's inbox immediately if they are online, or stored for retrieval when they next log in.

Reading a message that contains a gold attachment transfers the gold to the reader's character. Reading a message with an item attachment places the item into the reader's backpack (if space permits). Unread messages with attachments are preserved until the player can receive them.

Other modules can deliver mail programmatically via the exported SendMudMail(userId, senderName, message, gold, *item) function. The Auctions module uses this to notify offline winners and sellers.

An admin panel provides a view of all stored inboxes, with the ability to inspect and delete individual messages.

Room Tags

This module does not use any room tags.
