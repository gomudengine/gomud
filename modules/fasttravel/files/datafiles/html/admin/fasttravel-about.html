Fast Travel

Module overview and room tag reference.

Description

The Fast Travel module lets players teleport instantly between discovered fast travel stations. A station is discovered when a player types fasttravel or look fast travel in a room tagged with fast travel.

Once at a known station, fasttravel lists all other discovered destinations. fasttravel <destination> teleports the player there immediately. Travel may optionally require gold (GoldCost) or a consumable item (RequiredItemId). Certain item types or subtypes can be configured to block travel via DisallowedItemTypes and DisallowedItemSubtypes.

Per-player unlock data is persisted across sessions so players keep their discovered stations after logging out.

Room Tags

TagEffect
fast travel Marks the room as a fast travel station. Using the fasttravel command or examining the station here unlocks it for the player. When a player looks at the room, an alert is shown with instructions for using the network.
