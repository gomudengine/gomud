Storage API Reference

REST endpoints for viewing and managing player item storage.

GET /admin/api/v1/storage?user_id={id} List stored items for one user

Returns all items currently in the specified user's storage. The user_id query parameter is required. Works for both online and offline players.

Query parameters:

  • user_id (required) — the user's ID
curl -s -u admin:password \ "http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/storage?user_id=42"
200 Success

Returns user_id, username, and an items array. Each item contains:

  • index — 1-based position (matches in-game storage numbering)
  • item_id — the item spec ID
  • name — display name
  • short_id — unique instance identifier (used for deletion)
{ "success": true, "data": { "user_id": 42, "username": "alice", "items": [ { "index": 1, "item_id": 10001, "name": "steel sword", "short_id": "abc123" } ] } }
200 Empty storage
{ "success": true, "data": { "user_id": 42, "username": "alice", "items": [] } }
400 Bad Request

Returned when user_id is missing or not a positive integer.

{ "success": false, "data": { "error": "user_id query param is required" } }
401 Unauthorized
{ "success": false, "error": "unauthorized" }
DELETE /admin/api/v1/storage?user_id={id}&short_id={sid} Remove one item from a user's storage

Permanently removes the item with the matching short_id from the user's storage. The item is destroyed and cannot be recovered. Works for both online and offline players.

Query parameters:

  • user_id (required) — the user's ID
  • short_id (required) — the item's unique instance identifier (from the short_id field in the GET response)
curl -s -u admin:password \ -X DELETE \ "http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/storage?user_id=42&short_id=abc123"
200 Deleted

Returns confirmation with the deleted item's ID and display name.

{ "success": true, "data": { "deleted": true, "item_id": 10001, "item_name": "steel sword" } }
400 Bad Request

Returned when user_id or short_id is missing or invalid.

{ "success": false, "data": { "error": "user_id and short_id are required" } }
404 Not Found

No item matched the provided short_id in this user's storage.

{ "success": false, "data": { "error": "item not found" } }
401 Unauthorized
{ "success": false, "error": "unauthorized" }
