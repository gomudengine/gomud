Web Help

Module overview and room tag reference.

Description

The Web Help module exposes the game's in-game help system as public web pages. It provides two routes: /help lists all non-admin help topics grouped by category, and /help-details?search=<topic> renders the full content of a specific help topic with ANSI color codes converted to HTML.

These pages are publicly accessible without authentication, making the help content browsable outside the game client. Skill topics are automatically grouped under the skills category regardless of their declared category.

This module has no configuration keys, no persistent state, and no in-game commands.

Room Tags

This module does not use any room tags.
