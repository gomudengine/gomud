Cleanup

Module overview and room tag reference.

Description

The Cleanup module adds two commands for removing unwanted content from the world: trash and bury.

trash <item> permanently destroys an item from the player's backpack. If the ExperienceValue config key is set to a positive number, the player receives experience equal to the greater of that value or one-tenth of the item's gold value as a reward for cleaning up.

bury <corpse> removes a corpse from the current room. Both commands are also available as mob commands, allowing NPCs to clean up after combat automatically.

Room Tags

This module does not use any room tags.
