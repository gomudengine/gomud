CLI Bridge

Module overview and room tag reference.

Description

The CLI Bridge module gives players direct access to whitelisted command-line tools running on the server, rendered inside the MUD terminal via a pseudo-terminal (PTY). This is useful for in-game file editors, viewers, or other interactive tools that an operator wants to expose.

Players use the cli <tool> [args...] command to launch a session. The tool must be listed in the AllowedTools config key, and any file-path arguments must fall within the AllowedPaths list. Tab-completion is provided for both tool names and file paths within the data directory.

CLI sessions are only supported over raw TCP (telnet/SSH) connections. WebSocket clients receive an error message. The session ends when the tool exits, returning the player to normal game input.

Setting Enabled to false disables the command entirely without unloading the module.

Room Tags

This module does not use any room tags.
