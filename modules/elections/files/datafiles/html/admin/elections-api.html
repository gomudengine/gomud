Elections API Reference

REST endpoints for inspecting and administering zone election state: officials, tax rates, and coffer balances.

GET /admin/api/v1/elections/zones List all zones with election state

Returns every zone known to the server, sorted alphabetically. Each entry includes the current elected official (if any), the zone tax rate, and the coffer balance. Zones with no elected official have empty string values for official_name and official_title, and 0 for official_id.

The tax_rate field reflects the currently active rate. Zones with no explicitly set rate use the module default of 1.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/elections/zones
200 Success

Returns a zones array. Each entry contains:

  • zone - zone name
  • official_name - elected official's character name (empty if none)
  • official_title - the title they were elected to (empty if none)
  • official_id - the elected official's user ID (0 if none)
  • tax_rate - current zone tax rate, 0–100
  • coffer - current coffer balance in gold
{ "success": true, "data": { "zones": [ { "zone": "Frostfang", "official_name": "Sammy", "official_title": "Mayor of Frostfang", "official_id": 42, "tax_rate": 5, "coffer": 18400 }, { "zone": "The Wilds", "official_name": "", "official_title": "", "official_id": 0, "tax_rate": 1, "coffer": 0 } ] } }
401 Unauthorized
{ "success": false, "error": "unauthorized" }
PATCH /admin/api/v1/elections/zones/{zone} Override tax rate and/or coffer balance for a zone

Updates the tax rate and/or coffer balance for the named zone. At least one field must be provided. Changes take effect immediately.

Path parameters:

  • zone - the zone name (case-insensitive)

Request body fields (all optional, but at least one required):

  • tax_rate - new tax rate, integer 0–100
  • coffer - new coffer balance, integer 0–100000000

Set tax rate only:

curl -s -u admin:password \ -X PATCH \ -H "Content-Type: application/json" \ -d '{"tax_rate": 10}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/elections/zones/Frostfang

Set both tax rate and coffer balance:

curl -s -u admin:password \ -X PATCH \ -H "Content-Type: application/json" \ -d '{"tax_rate": 0, "coffer": 50000}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/elections/zones/Frostfang
200 Updated
{ "success": true, "data": { "zone": "frostfang", "updated": true } }
400 Bad Request

Returned when the body is malformed, no fields are provided, or a value is out of range.

{ "success": false, "data": { "error": "tax_rate must be between 0 and 100" } }
{ "success": false, "data": { "error": "at least one of tax_rate or coffer is required" } }
{ "success": false, "data": { "error": "coffer exceeds maximum of 100000000" } }
401 Unauthorized
{ "success": false, "error": "unauthorized" }
DELETE /admin/api/v1/elections/zones/{zone}/official Remove the elected official for a zone

Removes the elected official record for the named zone. The zone's tax rate and coffer balance are not affected. The player loses their title display and access to any elected officials only rooms in the zone. This action cannot be undone.

Path parameters:

  • zone - the zone name (case-insensitive)
curl -s -u admin:password \ -X DELETE \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/elections/zones/Frostfang/official
200 Removed
{ "success": true, "data": { "zone": "frostfang", "official_removed": true } }
400 Bad Request

Returned when the zone path parameter is empty.

{ "success": false, "data": { "error": "zone is required" } }
404 Not Found

Returned when the zone has no elected official to remove.

{ "success": false, "data": { "error": "no elected official for zone" } }
401 Unauthorized
{ "success": false, "error": "unauthorized" }
