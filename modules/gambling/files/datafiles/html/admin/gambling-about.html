Gambling

Module overview and room tag reference.

Description

The Gambling module adds two arcade-style gambling fixtures: a slot machine and a claw machine. Players interact with them using the play slots or play claw machine command while in a room that has the appropriate tag.

Slot machine — players pay a configurable cost per spin and win gold based on matching symbols. A portion of each losing spin accumulates in a jackpot that grows until someone hits the jackpot combination. The jackpot is persisted across restarts.

Claw machine — players pay a cost per attempt and have a configurable chance to win a prize item from the machine's prize pool.

Both fixtures can be examined with look slot machine or look claw machine to see a description and current state. The module also registers several prop items (dice, lucky coin, tarot deck, magic 8-ball, empty bottle, deck of cards) with scripted behaviors.

Room Tags

TagEffect
slots Marks the room as containing a slot machine. Either slots or slot machine activates the fixture. When a player looks at the room, an alert is shown inviting them to look at or play the slot machine.
slot machine Alias for the slots tag. Either value activates the same slot machine fixture in the room.
claw machine Marks the room as containing a claw machine. When a player looks at the room, an alert is shown inviting them to look at or play the claw machine.
