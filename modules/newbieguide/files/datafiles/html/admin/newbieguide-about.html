Newbie Guide

Module overview and room tag reference.

Description

The Newbie Guide module automatically spawns a helpful guide NPC to accompany new players as they explore the world. The guide appears whenever a low-level player moves to a new room (subject to a cooldown) and introduces itself, offering to create a portal back to Town Square on request.

The guide follows the player as a charmed mob. Once the player reaches the configured MaximumPlayerLevel (default 5), the guide bids them farewell and despawns. The guide mob ID is configurable via GuideMobId (default 38).

The guide will not spawn inside the character creation area (rooms 900–999) or respawn within the cooldown window (300 real seconds by default).

A PlayerGuideMobId function is exported so other modules can identify and interact with the guide mob.

Room Tags

This module does not use any room tags.
