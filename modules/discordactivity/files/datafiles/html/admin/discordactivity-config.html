Setup Instructions

Show step-by-step setup guide
  1. Go to discord.com/developers/applications and click New Application. Give it a name (e.g. "My MUD"). Note the Application ID shown on the General Information page.
  2. In the left sidebar, click Activities and enable the Activities feature for this application.
  3. Under Activities > URL Mappings, click Add a mapping and enter: 
    Prefix: /
Target: {{ .ActivityHost }}
    This tells Discord to proxy all Activity requests to your GoMud server.
  4. Copy the Application ID from the General Information page and paste it into the Application ID field above, then click Save.
  5. Your Activity will appear in Discord voice channels under the rocket icon once the application has been added to a server. Use the OAuth2 install URL from the Developer Portal to add it to your Discord server.
  6. HTTPS is required. Discord will not load an Activity served over plain HTTP. Configure Network.HttpsPort in the GoMud server settings and ensure your domain has a valid TLS certificate.
  7. Players connect using their normal MUD username and password through the terminal inside the Activity. No Discord OAuth login is required.
