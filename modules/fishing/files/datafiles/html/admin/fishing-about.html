fishing - About

Fishing module overview and configuration reference.

Overview

The fishing module lets players cast a line and catch items in specially tagged rooms. Each fishing attempt consumes one use of the chosen bait and lasts up to 15 rounds. Starting at round 3, a catch roll is made each round. The chance decays by 1% per round after round 3, so acting quickly gives the best odds.

Room Setup

To make a room fishable, add the tag fishing to it via the room editor.

Player Commands

  • fishing <baitname> — Begin fishing using the named bait item from your backpack.
  • fishing stop / fishing quit / fishing end — Reel in and stop fishing.
  • Typing fishing with no argument lists available bait in your backpack.

Requirements

  • The room must have the fishing tag.
  • The player must have a configured fishing rod item equipped in their weapon slot. Configure rod item IDs on the Config page. If no rod items are configured, the rod check is skipped.
  • The player must have a food/edible item in their backpack to use as bait. One use is consumed per attempt.

Catchable Items

Manage catchable items on the Catchables page. Each entry has:

  • Item — The item awarded on a successful catch.
  • Catch % — Percentage chance (1–100) that this item is eligible each roll.
  • Required Bait — If set, this item only appears when the player uses that specific bait. Leave blank for any bait.
  • Room IDs — If set, restricts this item to specific fishing rooms. Leave blank for all fishing rooms.

When a catch roll succeeds, all eligible items whose adjusted chance meets the roll are collected into a pool, and one is chosen at random.

Fishing Statmod

The fishing statmod (available on buffs and equipped items) adds to or subtracts from the catch chance each round. A value of +10 gives a 10% bonus; a value of -5 makes fishing harder.

Interruptions

Fishing is automatically cancelled when a player:

  • Moves to a different room.
  • Enters combat (aggro state changes).
  • Is knocked out (drops to 0 HP).
  • Disconnects from the server.
