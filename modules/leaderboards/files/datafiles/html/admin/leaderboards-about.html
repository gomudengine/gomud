Leaderboards

Module overview and room tag reference.

Description

The Leaderboards module maintains ranked lists of the top players across four categories: Gold (on-hand plus bank), Experience, Kills, and Exploration (total unique rooms visited). Leaderboards are recalculated at most once per real-time minute and consider both online and offline characters, including alt characters if the Alt Characters module is loaded.

Players can view the leaderboards in-game with the leaderboard command. A public web page at /leaderboards also displays the current standings for visitors who are not logged in.

The size of each leaderboard is individually configurable: GoldLBSize, ExperienceLBSize, KillsLBSize, and ExplorationLBSize. Setting any of these to 0 disables that particular leaderboard.

Room Tags

This module does not use any room tags.
