# Help for ~inspect~ (skill)

The ~inspect~ skills looks at items and reveals hidden details.

As you level up the ~inspect~ skill, new things can become revealed.

## Usage:

(Lvl 1) ~inspect [item_name]~ Reveals the type and value of items.  
(Lvl 2) ~inspect [item_name]~ Reveals weapon damage or uses an item has left.  
(Lvl 3) ~inspect [item_name]~ Reveals any stat modifiers an item has.  
(Lvl 4) ~inspect [item_name]~ Reveals special magical properties like elemental effects.
