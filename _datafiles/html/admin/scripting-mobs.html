Mob Scripts

Event-driven JavaScript hooks that give NPCs intelligent, reactive behavior.

Script Path

Mob scripts live in a scripts/ subdirectory next to the mob YAML file:

Mob definition _datafiles/world/default/mobs/frostfang/2-guard.yaml
Default mob script _datafiles/world/default/mobs/frostfang/scripts/2-guard.js
Script-tagged variant (scripttag: hungry) _datafiles/world/default/mobs/frostfang/scripts/2-guard-hungry.js

When a room's spawninfo sets a scripttag, that tag is appended to the mob's base script name with a hyphen. This lets you create context-specific variants of the same mob without duplicating the mob definition.

Script State

Global variables in a mob script are shared across all instances of that mob type. For per-instance data, use mob.SetTempData(key, value) and mob.GetTempData(key).

Event Hooks

onLoad(mob ActorObject)

Called within one round of a mob instance being created, or sooner if players are present. Ideal for expensive initialization. Given a longer execution timeout than other hooks.

ArgumentTypeDescription
mobActorObjectThe mob instance being initialized.
function onLoad(mob) {
    mob.SetTempData('greeting_count', 0);
}
onIdle(mob ActorObject, room RoomObject)

Called each round when the mob is not in combat and not walking home. Return false to allow the mob's default idle behaviors (look for trouble, first aid, etc.) to run after your script. Return true to suppress them.

Return value: true = suppress default idle checks. false or nothing = allow them.
ArgumentTypeDescription
mobActorObjectThe mob.
roomRoomObjectThe room the mob is in.
function onIdle(mob, room) {
    if (Math.random() < 0.05) {
        mob.Command('say Lovely weather today.');
    }
    return false; // still do normal idle checks
}
onGive(mob ActorObject, room RoomObject, eventDetails object)

Called when an item or gold is given to the mob via the give command. Return true to prevent the mob from auto-equipping the item (if applicable).

Return value: true = prevent auto-equip. false or nothing = allow it.
ArgumentTypeDescription
mobActorObjectThe mob receiving the gift.
roomRoomObjectThe current room.
eventDetails.sourceIdintuserId or mobInstanceId of the giver.
eventDetails.sourceTypestring"user" or "mob".
eventDetails.goldintAmount of gold given (0 if an item was given).
eventDetails.itemItemObjectThe item given (null if gold was given).
function onGive(mob, room, eventDetails) {
    if (eventDetails.gold > 0) {
        mob.Command('say Thank you for the gold!');
    } else if (eventDetails.item) {
        mob.Command('say What a curious trinket.');
    }
    return true; // don't auto-equip
}
onShow(mob ActorObject, room RoomObject, eventDetails object)

Called when a player uses the show command to display an item to the mob.

ArgumentTypeDescription
mobActorObjectThe mob being shown the item.
roomRoomObjectThe current room.
eventDetails.sourceIdintuserId or mobInstanceId of who showed it.
eventDetails.sourceTypestring"user" or "mob".
eventDetails.itemItemObjectThe item being shown.
function onShow(mob, room, eventDetails) {
    if (eventDetails.item) {
        mob.Command('say Interesting item you have there.');
    }
}
onAsk(mob ActorObject, room RoomObject, eventDetails object)

Called when a player asks the mob something. Return false to make the mob respond with a generic rejection (e.g. "the guard shakes their head").

Return value: false = generic rejection. true or nothing = no default response.
ArgumentTypeDescription
mobActorObjectThe mob being asked.
roomRoomObjectThe current room.
eventDetails.sourceIdintuserId or mobInstanceId of the asker.
eventDetails.sourceTypestring"user" or "mob".
eventDetails.askTextstringThe question text.
function onAsk(mob, room, eventDetails) {
    var q = eventDetails.askText.toLowerCase();
    if (q.indexOf('password') !== -1) {
        mob.Command('say The password is swordfish.');
        return true;
    }
    return false; // generic rejection for everything else
}
onCommand(cmd string, rest string, mob ActorObject, room RoomObject, eventDetails object)

Called whenever anyone in the room types any command. Fires before the room's onCommand. If multiple mobs in the room have this hook, one returning true will prevent downstream mobs from being called.

Return value: true = consumed. false or nothing = pass through.
ArgumentTypeDescription
cmdstringThe command word.
reststringEverything after the command word.
mobActorObjectThis mob.
roomRoomObjectThe current room.
eventDetails.sourceIdintuserId or mobInstanceId of the command sender.
eventDetails.sourceTypestring"user" or "mob".
function onCommand(cmd, rest, mob, room, eventDetails) {
    if (cmd === 'attack' && eventDetails.sourceType === 'user') {
        mob.Command('say You dare attack me?!');
    }
    return false;
}
onCommand_{command}(rest string, mob ActorObject, room RoomObject, eventDetails object)

Targeted variant of onCommand. Replace {command} with the exact command word to intercept. When defined, the generic onCommand is not called for that command.

Return value: true = consumed. false or nothing = pass through.
ArgumentTypeDescription
reststringEverything after the command word.
mobActorObjectThis mob.
roomRoomObjectThe current room.
eventDetails.sourceIdintuserId or mobInstanceId of the sender.
eventDetails.sourceTypestring"user" or "mob".
function onCommand_greet(rest, mob, room, eventDetails) {
    if (eventDetails.sourceType === 'user') {
        mob.Command('say Greetings, traveler!');
        return true;
    }
    return false;
}
onHurt(mob ActorObject, room RoomObject, eventDetails object)

Called when the mob takes damage from an attack.

ArgumentTypeDescription
mobActorObjectThe mob taking damage.
roomRoomObjectThe current room.
eventDetails.sourceIdintuserId or mobInstanceId of the attacker.
eventDetails.sourceTypestring"user" or "mob".
eventDetails.damageintHitpoints of damage dealt.
eventDetails.critbooltrue if the hit was a critical strike.
function onHurt(mob, room, eventDetails) {
    if (mob.GetHealthPct() < 0.25) {
        mob.Command('say I yield! Mercy!');
    }
}
onDie(mob ActorObject, room RoomObject, eventDetails object)

Called when the mob dies — once for each player that dealt damage to it.

ArgumentTypeDescription
mobActorObjectThe dying mob.
roomRoomObjectThe current room.
eventDetails.sourceIdintuserId or mobInstanceId that killed it.
eventDetails.sourceTypestring"user" or "mob".
eventDetails.attackerCountintTotal number of users who dealt damage.
function onDie(mob, room, eventDetails) {
    if (eventDetails.sourceType === 'user') {
        var killer = GetUser(eventDetails.sourceId);
        if (killer) {
            killer.GiveQuest('5-end');
        }
    }
}
onPath(mob ActorObject, room RoomObject, eventDetails object)

Called during pathfinding as the mob moves. Return true to end the current path. You can safely start a new path with mob.Command('pathto 123') before returning true.

Return value: true = stop pathing. false or nothing = continue.
ArgumentTypeDescription
mobActorObjectThe mob pathing.
roomRoomObjectThe current room.
eventDetails.statusstring"start", "waypoint", or "end".
function onPath(mob, room, eventDetails) {
    if (eventDetails.status === 'end') {
        mob.Command('say I have arrived.');
    }
    return false;
}
onPlayerDowned(mob ActorObject, user ActorObject, room RoomObject)

Called when a player is downed (incapacitated but not killed) by this mob. Return true to ensure this hook only fires once per mob type in the room (preventing multiple guards from all reacting).

Return value: true = fire once per mob type. false or nothing = fire for every instance.
ArgumentTypeDescription
mobActorObjectThe mob that downed the player.
userActorObjectThe downed player.
roomRoomObjectThe current room.
function onPlayerDowned(mob, user, room) {
    mob.Command('say Stay down if you value your life.');
    return true; // only one guard reacts
}
