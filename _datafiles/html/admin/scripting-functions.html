Global utility, messaging, and panel layout functions available in all script types. For object APIs see Object Reference.

Utility Functions

Global

General-purpose global functions available in all script types.

UtilGetRoundNumber()int

Returns the current game round number (always incrementing).

UtilFindMatchIn(search string, items []any)object

Searches a list for an exact or close match (minimum 3 characters). Returns an object with:

PropertyTypeDescription
foundboolTrue if any match was found.
exactstringExact match string (empty if none).
closestringClose match string (empty if none).
UtilGetSecondsToRounds(seconds int)int

Converts a number of real seconds to game rounds.

UtilGetMinutesToRounds(minutes int)int

Converts a number of real minutes to game rounds.

UtilGetSecondsToTurns(seconds int)int

Converts a number of real seconds to turns.

UtilGetMinutesToTurns(minutes int)int

Converts a number of real minutes to turns.

UtilStripPrepositions(input string)string

Strips common prepositions and grammatical noise from a string (e.g. "into", "to", "from", "the", "my").

UtilDiceRoll(diceQty int, diceSides int)int

Simulates a dice roll and returns the result.

ArgumentTypeDescription
diceQtyintNumber of dice to roll.
diceSidesintNumber of sides per die.
var dmg = UtilDiceRoll(2, 6); // 2d6
local dmg = UtilDiceRoll(2, 6) -- 2d6
UtilGetTime()object

Returns the current in-game time as an object with:

PropertyTypeDescription
DayintTotal days elapsed.
HourintCurrent hour (12-hour).
Hour24intCurrent hour (24-hour).
MinuteintCurrent minute.
AmPmstring"AM" or "PM".
NightboolTrue if currently nighttime.
DayStartintHour when day begins (24-hour).
NightStartintHour when night begins (24-hour).
UtilIsDay()bool

Returns true if it is currently daytime.

UtilSetTimeDay()void

Sets the game time to 1 round before day breaks.

UtilSetTime(hour int, minutes int)void

Sets the game time to a specific hour and minute (24-hour format). Example: 5:30 PM = UtilSetTime(17, 30).

UtilLocateUser(search int|string)int

Returns the room ID of the specified user, or 0 if not found.

ArgumentTypeDescription
searchint | stringUsername or userId to find.
UtilApplyColorPattern(input string, patternName string [, wordsOnly bool])string

Applies a named color pattern to a string and returns the colorized result.

ArgumentTypeDescription
inputstringPlain text to colorize.
patternNamestringColor pattern name, e.g. "rainbow".
wordsOnlybool (optional)If true, change color only at word boundaries.
UtilGetConfig()object

Returns a config object with properties from the server configuration YAML.

Messaging Functions

Global

Global functions for sending messages to users, rooms, or the entire server.

SendBroadcast(message string)void

Sends a message to every connected player on the server.

SendUserMessage(userId int, message string)void

Sends a message to a specific user by ID.

ArgumentTypeDescription
userIdintThe recipient's user ID.
messagestringThe message to send.
SendRoomMessage(roomId int, message string [, excludeUserIds…])void

Sends a message to all players in the specified room.

ArgumentTypeDescription
roomIdintTarget room ID.
messagestringThe message to send.
excludeUserIdsint… (optional)User IDs to exclude.
SendRoomExitsMessage(roomId int, message string, isQuiet bool [, excludeUserIds…])void

Sends a message to all rooms with an exit leading to the specified room.

ArgumentTypeDescription
roomIdintThe source room ID.
messagestringThe message to send.
isQuietboolIf true, only players with superior hearing receive it.
excludeUserIdsint… (optional)User IDs to exclude.

Panel Layout Functions

Global

Build richly formatted terminal displays made up of titled boxes arranged side by side or stacked vertically — the same system used for the in-game character status screen.

PanelLayoutNew([opts object])PanelLayoutObject

Creates a blank panel layout entirely in script. All opts properties are optional.

OptionDefaultDescription
border"full""full" — every row has side borders. "open" — only first and last rows.
charset"single""single" ┌─┐, "double" ╔═╗, "rounded" ╭─╮, or 8-char literal.
gap1Spaces between panels placed side by side.
margin0Spaces prepended to every output line.
var layout = PanelLayoutNew({ border: 'full', charset: 'rounded', gap: 1, margin: 1 });
local layout = PanelLayoutNew({ border = "full", charset = "rounded", gap = 1, margin = 1 })
PanelLayoutLoad(name string)PanelLayoutObject

Loads a panel layout from a YAML file under _datafiles/panel-layouts/. Throws a JS error if the file doesn't exist.

ArgumentTypeDescription
namestringPath relative to panel-layouts/, without extension. E.g. "character/status".
PanelLayoutObject.AddSlot()SlotObject

Adds a new vertical slot (column) to the layout. Slots are rendered side by side. Returns a SlotObject for adding rows.

PanelLayoutObject.Panel(id string)PanelObject

Returns the named panel for configuration and data population. Throws if the ID doesn't exist.

PanelLayoutObject.Render()string

Renders the entire layout into a terminal string ready to send to a player.

SlotObject.AddRow([ids []string])SlotObject

Adds a horizontal row of panels to the slot. Returns the slot for chaining.

var slot = layout.AddSlot();
slot.AddRow(['health', 'mana']);  // two panels side by side
slot.AddRow(['gold']);            // one panel underneath
local slot = layout:AddSlot()
slot:AddRow({"health", "mana"})  -- two panels side by side
slot:AddRow({"gold"})            -- one panel underneath
PanelObject.SetTitle(title string)PanelObject

Sets the panel's title. May contain ANSI tags. Returns panel for chaining.

PanelObject.SetCharset(name string)PanelObject

Overrides the border character set for this panel only. Returns panel for chaining.

PanelObject.SetMinWidth(n int)PanelObject

Sets the minimum inner content width in characters. Returns panel for chaining.

PanelObject.SetColumns(n int)PanelObject

Sets how many label+value pairs per rendered line. 1 (default) or 2. Returns panel for chaining.

PanelObject.SetColumnGap(n int)PanelObject

Sets the number of spaces between columns when using 2-column mode. Default 2. Returns panel for chaining.

PanelObject.Add(fullLabel, shortLabel, value string)PanelObject

Appends a label+value row to the panel. The renderer uses fullLabel when it fits, falling back to shortLabel when the panel is too narrow. Returns panel for chaining.

layout.Panel('info')
    .Add('<ansi fg="yellow">Health:</ansi>', '<ansi fg="yellow">HP:</ansi>', '142/200')
    .Add('<ansi fg="yellow">Mana:  </ansi>', '<ansi fg="yellow">MP:</ansi>', '80/100');
layout:Panel("info")
    :Add("<ansi fg=\"yellow\">Health:</ansi>", "<ansi fg=\"yellow\">HP:</ansi>", "142/200")
    :Add("<ansi fg=\"yellow\">Mana:  </ansi>", "<ansi fg=\"yellow\">MP:</ansi>", "80/100")
PanelObject.AddBlank()PanelObject

Appends an empty spacer row for visual separation. Returns panel for chaining.

Full example — two-panel status display 
function onCommand_status(rest, user, room) {
    var layout = PanelLayoutNew({ border: 'full', charset: 'single', gap: 1, margin: 1 });

    var slot = layout.AddSlot();
    slot.AddRow(['info', 'stats']);

    layout.Panel('info')
        .SetTitle(' .:Character ')
        .SetMinWidth(24)
        .Add('Name:  ', 'N:', user.GetCharacterName(false))
        .Add('Level: ', 'Lvl:', String(user.GetLevel()))
        .Add('Health:', 'HP:', user.GetHealth() + '/' + user.GetHealthMax())
        .Add('Mana:  ', 'MP:', user.GetMana() + '/' + user.GetManaMax());

    layout.Panel('stats')
        .SetTitle(' .:Attributes ')
        .SetMinWidth(28)
        .SetColumns(2)
        .Add('Strength: ', 'Str:', String(user.GetStat('strength')))
        .Add('Vitality: ', 'Vit:', String(user.GetStat('vitality')))
        .Add('Speed:    ', 'Spd:', String(user.GetStat('speed')))
        .Add('Mysticism:', 'Mys:', String(user.GetStat('mysticism')));

    user.SendText(layout.Render());
    return true;
}
function onCommand_status(rest, user, room)
    local layout = PanelLayoutNew({ border = "full", charset = "single", gap = 1, margin = 1 })

    local slot = layout:AddSlot()
    slot:AddRow({"info", "stats"})

    layout:Panel("info")
        :SetTitle(' <ansi fg="black-bold">.:</ansi><ansi fg="20">Character</ansi> ')
        :SetMinWidth(24)
        :Add('<ansi fg="yellow">Name:  </ansi>', '<ansi fg="yellow">N:</ansi>', user:GetCharacterName(false))
        :Add('<ansi fg="yellow">Level: </ansi>', '<ansi fg="yellow">Lvl:</ansi>', tostring(user:GetLevel()))
        :Add('<ansi fg="yellow">Health:</ansi>', '<ansi fg="yellow">HP:</ansi>', user:GetHealth() .. "/" .. user:GetHealthMax())
        :Add('<ansi fg="yellow">Mana:  </ansi>', '<ansi fg="yellow">MP:</ansi>', user:GetMana() .. "/" .. user:GetManaMax())

    layout:Panel("stats")
        :SetTitle(' <ansi fg="black-bold">.:</ansi><ansi fg="20">Attributes</ansi> ')
        :SetMinWidth(28)
        :SetColumns(2)
        :Add('<ansi fg="yellow">Strength: </ansi>', '<ansi fg="yellow">Str:</ansi>', tostring(user:GetStat("strength")))
        :Add('<ansi fg="yellow">Vitality: </ansi>', '<ansi fg="yellow">Vit:</ansi>', tostring(user:GetStat("vitality")))
        :Add('<ansi fg="yellow">Speed:    </ansi>', '<ansi fg="yellow">Spd:</ansi>', tostring(user:GetStat("speed")))
        :Add('<ansi fg="yellow">Mysticism:</ansi>', '<ansi fg="yellow">Mys:</ansi>', tostring(user:GetStat("mysticism")))

    user:SendText(layout:Render())
    return true
end
