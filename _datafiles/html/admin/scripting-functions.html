ActorObject

Users & Mobs

Represents a player or NPC. Obtained from event hook arguments, or via GetUser(userId) / GetMob(mobInstanceId). Used in room, mob, item, buff, spell, and pet scripts.

ActorNames(actors []ActorObject) string

Returns a formatted comma-separated list of actor names ending with "and". Example: "Tim, Jim and Henry".

ArgumentTypeDescription
actors[]ActorObjectArray of ActorObjects to format.
GetUser(userId int) ActorObject

Retrieves an ActorObject for a given userId.

ArgumentTypeDescription
userIdintThe user ID to retrieve.
GetMob(mobInstanceId int [, createIfMissing bool]) ActorObject

Retrieves an ActorObject for a given mob instance ID. If createIfMissing is true and the mob isn't found, it will be created and returned.

ArgumentTypeDescription
mobInstanceIdintThe mob instance ID to retrieve.
createIfMissingbool (optional)If true, create the mob if not found.
ActorObject.UserId()int

Returns the userId. Only meaningful for user actors — returns 0 for mobs.

ActorObject.InstanceId()int

Returns the mob instance ID. Only meaningful for mob actors — returns 0 for users.

ActorObject.MobTypeId()int

Returns the base mob ID used to spawn new instances. Only meaningful for mob actors — returns 0 for users.

ActorObject.ShorthandId()string

Returns the shorthand targeting string — @123 for users or #122 for mobs. Useful in Command() calls for precise targeting.

ActorObject.GetCharacterName(wrapInTags bool)string

Returns the actor's name. When wrapInTags is true, wraps it in ANSI color tags using username or mobname color.

ArgumentTypeDescription
wrapInTagsboolIf true, wrap name in ANSI color tags.
ActorObject.SetCharacterName(newName string)void

Renames the mob or player.

ArgumentTypeDescription
newNamestringThe new name.
ActorObject.GetRace()string

Returns the race name of the actor, e.g. "Human", "Elf", "Rodent".

ActorObject.GetSize()string

Returns "small", "medium", or "large".

ActorObject.GetLevel()int

Returns the actor's current level.

ActorObject.GetRoomId()int

Returns the room ID the actor is currently in.

ActorObject.GetStat(statName string)int

Returns the value of a named stat.

ArgumentTypeDescription
statNamestringStat name: "strength", "speed", "smarts", "vitality", "mysticism", "perception".
ActorObject.GetStatMod(statModName string)int

Returns the total value of a special stat modifier from worn items and buffs combined.

ArgumentTypeDescription
statModNamestringModifier name, e.g. "strength" or "tame".
ActorObject.GetHealth()int

Returns current health points.

ActorObject.SetHealth(amt int)void

Sets health to a specific amount. Capped at the actor's maximum health.

ArgumentTypeDescription
amtintHitpoints to set.
ActorObject.GetHealthMax()int

Returns the actor's maximum health.

ActorObject.GetHealthPct()float

Returns current health as a percentage (0.0–1.0).

ActorObject.AddHealth(amt int)int

Adds (or subtracts) health. Returns the actual amount changed.

ArgumentTypeDescription
amtintPositive to heal, negative to damage.
ActorObject.GetMana()int

Returns current mana.

ActorObject.GetManaMax()int

Returns the actor's maximum mana.

ActorObject.GetManaPct()float

Returns current mana as a percentage (0.0–1.0).

ActorObject.AddMana(amt int)int

Adds (or subtracts) mana. Returns the actual amount changed.

ArgumentTypeDescription
amtintPositive to restore, negative to drain.
ActorObject.SendText(msg string)void

Sends a message directly to this actor (only they see it).

ArgumentTypeDescription
msgstringThe message to send. May contain ANSI tags.
ActorObject.PlaySound(soundId string, category string)void

Plays a sound for this actor. Only affects user actors — silently ignored for mobs. soundId must match an identifier defined in audio.yaml (e.g. "levelup"). category groups related sounds for client-side volume control (e.g. "combat", "other").

ArgumentTypeDescription
soundIdstringIdentifier from audio.yaml (e.g. "levelup", "hit-self").
categorystringClient volume category (e.g. "combat", "movement", "other").
ActorObject.PlayMusic(musicFileOrId string)void

Changes the background music for this actor. Only affects user actors. musicFileOrId may be a filepath (e.g. "static/audio/music/intro.mp3") or a sound identifier from audio.yaml whose filepath resolves to a music file. Pass "Off" to stop music.

ArgumentTypeDescription
musicFileOrIdstringMusic filepath, audio.yaml identifier, or "Off" to stop.
ActorObject.Command(cmd string [, waitTurns int])void

Forces the actor to execute a command as if they typed it. Extremely powerful — can trigger any in-game command including say, attack, pathto, etc.

ArgumentTypeDescription
cmdstringThe full command string, e.g. "say Hello!" or "attack goblin".
waitTurnsint (optional)Turns to wait before executing (NOT rounds).
mob.Command('say Welcome, traveler!');
mob.Command('pathto 150', 2); // move after 2 turns
ActorObject.MoveRoom(destRoomId int)void

Quietly moves the actor to a different room (no movement messages shown).

ArgumentTypeDescription
destRoomIdintDestination room ID.
ActorObject.Sleep(seconds int) void Mob only

Forces a mob to wait the specified number of seconds before executing any additional behaviors. Has no effect on players.

ArgumentTypeDescription
secondsintSeconds to pause.
ActorObject.AddGold(amt int [, bankAmt int])void

Adjusts the actor's gold (and optionally bank gold) by the given amounts.

ArgumentTypeDescription
amtintPositive or negative gold to add to carried gold.
bankAmtint (optional)Positive or negative gold to add to bank gold.
ActorObject.GiveItem(item ItemObject)void

Places an item into the actor's backpack. Can be called multiple times to duplicate an item.

ArgumentTypeDescription
itemItemObjectThe item to give.
ActorObject.TakeItem(item ItemObject)void

Removes an item from the actor's backpack.

ArgumentTypeDescription
itemItemObjectThe item to remove.
ActorObject.UpdateItem(item ItemObject)void

Saves changes to an item already in the actor's backpack. This is the only way to persist modifications to a backpack item.

ArgumentTypeDescription
itemItemObjectThe modified item to save.
ActorObject.HasItemId(itemId int [, excludeWorn bool])bool

Returns true if the actor has the specified item ID in their backpack.

ArgumentTypeDescription
itemIdintThe item ID to check for.
excludeWornbool (optional)If true, ignore equipped items.
ActorObject.GetBackpackItems()[]ItemObject

Returns an array of all items in the actor's backpack. See ItemObject for available methods.

ActorObject.HasBuff(buffId int)bool

ArgumentTypeDescription
buffIdintThe buff ID to check.
ActorObject.GiveBuff(buffId int, source string)void

Applies a buff to the actor.

ArgumentTypeDescription
buffIdintThe buff ID to apply.
sourcestringSource label: "item", "spell", "trap", "curse", or empty.
ActorObject.RemoveBuff(buffId int)void

Removes a buff without triggering its onEnd script.

ArgumentTypeDescription
buffIdintThe buff ID to remove.
ActorObject.HasBuffFlag(buffFlag string)bool

Returns true if the actor has any buff with the specified flag.

ArgumentTypeDescription
buffFlagstringThe buff flag name to check.
ActorObject.CancelBuffWithFlag(buffFlag string)bool

Cancels all buffs that have the specified flag. Returns true if at least one was found and cancelled.

ArgumentTypeDescription
buffFlagstringThe buff flag to match.
ActorObject.HasQuest(questId string)bool

Returns true if the actor has the specified quest or quest step.

ArgumentTypeDescription
questIdstringQuest identifier, e.g. "3-start" or "3-end".
ActorObject.GiveQuest(questId string)void

Grants a quest or quest progress step to the actor.

ArgumentTypeDescription
questIdstringQuest identifier, e.g. "3-start".
ActorObject.TrainSkill(skillName string, skillLevel int)void

Sets the actor's skill level if it's greater than their current level.

ArgumentTypeDescription
skillNamestringSkill name, e.g. "map", "backstab".
skillLevelintTarget skill level.
ActorObject.GetSkillLevel(skillName string)int

Returns the current level of the named skill, or 0 if not learned.

ArgumentTypeDescription
skillNamestringSkill name to check.
ActorObject.HasSpell(spellId string)bool

Returns true if the actor knows the specified spell.

ArgumentTypeDescription
spellIdstringThe spell ID to check.
ActorObject.LearnSpell(spellId string)bool

Adds the spell to the actor's spellbook. Returns true if learned, false if already known.

ArgumentTypeDescription
spellIdstringThe spell ID to learn.
ActorObject.GetAlignment()int

Returns the numeric alignment value from -100 (Unholy) to +100 (Holy).

ActorObject.GetAlignmentName()string

Returns the alignment label, e.g. "Holy", "Neutral", "Unholy".

ActorObject.ChangeAlignment(alignmentChange int)void

Adjusts alignment by a relative amount. Result is capped at -100 to +100.

ArgumentTypeDescription
alignmentChangeintDelta from -200 to +200.
ActorObject.IsCharmed([userId1, userId2, …])bool

Returns true if the actor is charmed. If user IDs are provided, returns true only if charmed by one of those users.

ActorObject.GetCharmedUserId()int

Returns the userId that charmed this actor, or 0 if not charmed.

ActorObject.CharmSet(userId int, charmRounds int [, onRevertCmd…])void

Charms the mob to a user for a set number of rounds (-1 for unlimited). Optional revert commands run when charm expires.

ArgumentTypeDescription
userIdintThe user to charm the mob to.
charmRoundsintDuration in rounds, or -1 for unlimited.
onRevertCmdstring (optional, repeatable)Commands to execute when charm expires.
ActorObject.CharmRemove()void

Immediately removes any charm effect without triggering expiration commands.

ActorObject.CharmExpire()void

Forces the current charm to expire (runs expiration commands).

ActorObject.GetCharmCount()int

Returns the number of creatures currently charmed by this actor.

ActorObject.GetMaxCharmCount()int

Returns the maximum number of creatures this actor can charm simultaneously.

ActorObject.GetParty([excludeSelf bool])PartyObject

Returns a PartyObject for the actor's entire party and charmed mobs.

ArgumentTypeDescription
excludeSelfbool (optional)If true, exclude this actor from the party object.
ActorObject.GetPartyPresent([excludeSelf bool])PartyObject

Returns a PartyObject for party members currently in the same room.

ActorObject.GetPartyMissing()PartyObject

Returns a PartyObject for party members not in the same room.

ActorObject.IsTameable()bool

Returns true if this actor can be tamed as a pet.

ActorObject.GetPet()PetObject | null

Returns the PetObject for this actor, or null if they have no pet. Only meaningful for user actors.

The returned PetObject is a live reference — mutations take effect immediately.

var pet = actor.GetPet();
if (pet !== null) {
    room.SendText(pet.NameSimple() + ' is here!');
}
ActorObject.GetTameMastery()object

Returns an object where keys are mob IDs and values are tame mastery levels.

ArgumentTypeDescription

ArgumentTypeDescription

ActorObject.GetMiscCharacterData(key string)any

ArgumentTypeDescription

ActorObject.GetLastInputRound()int

RoomObject

Rooms

Represents a room. Available in all script types. Also includes global room-creation and map functions.

GetRoom(roomId int)RoomObject

Retrieves a RoomObject for the given room ID.

RoomObject.RoomId()int

Returns the room's ID.

RoomObject.SendText(msg string [, excludeUserIds…])void

Sends a message to all players in the room. Optionally exclude specific user IDs.

ArgumentTypeDescription
msgstringMessage to send. May contain ANSI tags.
excludeUserIdsint… (optional)One or more user IDs to exclude.
RoomObject.SendTextToExits(msg string, isQuiet bool [, excludeUserIds…])void

Sends a message to all rooms that have an exit leading to this room.

ArgumentTypeDescription
msgstringMessage to send.
isQuietboolIf true, only players with superior hearing receive it.
excludeUserIdsint… (optional)User IDs to exclude.
RoomObject.SetTempData(key string, value any)void

Stores temporary data on the room (lost when room unloads from memory). Useful for short-term state like a triggered switch.

RoomObject.GetTempData(key string)any

Retrieves temporary room data. Returns null if not set.

RoomObject.SetPermData(key string, value any)void

Stores permanent data on the room (saved even when room unloads). Useful for persistent state like leaderboards or ownership.

RoomObject.GetPermData(key string)any

Retrieves permanent room data.

RoomObject.GetItems()[]ItemObject

Returns all items on the floor of the room.

RoomObject.PlaySound(soundId string, category string [, excludeUserIds…])void

Plays a sound for all players currently in the room. soundId must match an identifier defined in audio.yaml (e.g. "room-enter"). category groups related sounds for client-side volume control. Optionally exclude specific user IDs from receiving the sound.

ArgumentTypeDescription
soundIdstringIdentifier from audio.yaml (e.g. "room-enter", "change").
categorystringClient volume category (e.g. "combat", "movement", "other").
excludeUserIdsint… (optional)User IDs that should not receive the sound.
RoomObject.GetStashItems()[]ItemObject

Returns all hidden (stashed) items in the room — not visible to players in normal room descriptions.

RoomObject.SpawnItem(itemId int, inStash bool)[]ItemObject

Spawns an item in the room.

ArgumentTypeDescription
itemIdintItem ID to spawn.
inStashboolIf true, spawn hidden in the stash.
RoomObject.DestroyItem(item ItemObject)void

Destroys an item from the room floor.

RoomObject.RepeatSpawnItem(itemId int, roundInterval int [, containerName string])bool

Sets up automatic item respawning. The item respawns after the given interval when removed from the room.

ArgumentTypeDescription
itemIdintItem to respawn.
roundIntervalintRounds to wait before respawning.
containerNamestring (optional)Container to spawn into.
RoomObject.GetMob(mobId int [, createIfMissing bool])ActorObject

Returns the first mob in the room matching the given mob type ID (NOT instance ID). Optionally spawns one if not found.

ArgumentTypeDescription
mobIdintMob type ID to find.
createIfMissingbool (optional)If true, spawn if not found.
RoomObject.GetMobs([mobId int])[]ActorObject

Returns all mobs in the room. Optionally filter by mob type ID.

RoomObject.GetPlayers()[]ActorObject

Returns all player actors in the room.

RoomObject.GetAllActors()[]ActorObject

Returns all actors (players and mobs) in the room.

RoomObject.SpawnMob(mobId int)ActorObject

Spawns a new instance of the mob type and returns its ActorObject.

ArgumentTypeDescription
mobIdintMob type ID to spawn (NOT instance ID).
RoomObject.GetContainers()[]string

Returns the names of all containers in the room.

RoomObject.HasTag(tag string)bool

RoomObject.UnsetTag(tag string)void

ItemObject

Items

Represents an item instance. Obtained from actor backpacks, room floors, or CreateItem(itemId).

CreateItem(itemId int)ItemObject

Creates a new item instance from the given item ID and returns it. Use ActorObject.GiveItem() to put it in a player's backpack.

ItemObject.ItemId()int

Returns the item's definition ID.

ItemObject.DisplayName([plainFormat bool])string

Returns the item's display name. Pass true for plain text without color tags.

ItemObject.NameSimple()string

Returns the simple base name, e.g. "Axe" for a "Glowing Battleaxe".

ItemObject.NameComplex()string

Returns the full complex name including enchantments, e.g. "Glowing Battleaxe +2 [c]".

ItemObject.ShorthandId()string

Returns the shorthand targeting string for this item, e.g. "!123".

ItemObject.GetUsesLeft()int

Returns the number of uses remaining on the item.

ItemObject.SetUsesLeft(amount int)int

Sets the remaining uses to a specific number.

ItemObject.AddUsesLeft(amount int)int

Adds (or subtracts) uses. Positive to add, negative to remove.

ItemObject.GetLastUsedRound()uint64

Returns the round number when this item was last used.

ItemObject.MarkLastUsed([clear bool])uint64

Sets the last-used round to the current round. Pass true to clear it back to zero.

ItemObject.SetTempData(key string, value any)void

Stores temporary data on the item (not saved when despawned).

ItemObject.GetTempData(key string)any

Retrieves temporary item data.

ItemObject.Rename(newName string [, displayNameOrStyle string])void

Renames the item. Optionally provide a fancy display name with ANSI tags or a color pattern like ":flame".

ItemObject.Redescribe(newDescription string)void

Changes the item's description text.

PetObject

Pets

Represents a player's companion pet. Obtained via ActorObject.GetPet(). Returns null if the actor has no pet. This is a live reference — mutations take effect immediately.

PetObject.Type()string

Returns the pet's type identifier, e.g. "dog", "cat", "owl".

PetObject.Name()string

Returns the full display name including ANSI color tags and hunger indicator (e.g. (Hungry)). Use for output to players.

PetObject.NameSimple()string

Returns the plain text name with no color or hunger indicator. Use for comparisons and concatenation.

PetObject.SetName(name string)void

Renames the pet. Pass an empty string to clear the custom name and revert to the type identifier.

PetObject.Level()int

Returns the pet's current level (1–10). Increases when well-fed, decreases when starving.

PetObject.Food()string

Returns the hunger state as a string.

ValueMeaning
"Starving"Food level 0 — will lose a level at next daily tick.
"Hungry"Food level 1.
"Satisfied"Food level 2.
"Full"Food level 3 — will gain a level at next daily tick.
PetObject.FoodLevel()int

Returns the raw hunger level as an integer: 0 (Starving) through 3 (Full).

PetObject.Feed()void

Increases hunger level by one step (max 3 / Full). Does not consume any item from inventory.

PetObject.Starve()void

Decreases hunger level by one step (min 0 / Starving).

PetObject.GetStatMod(statName string)int

Returns the stat modifier the pet grants its owner for the named stat at its current level.

PetObject.GetCapacity()int

Returns the number of items the pet can carry at its current level. 0 if the pet type has no carry ability.

PetObject.ItemCount()int

Returns the number of items the pet is currently carrying.

PetObject.HasScript()bool

Returns true if this pet type has a script file on disk.

PetObject.IsMissing()bool

Returns true when the pet is temporarily absent (MissingCountdown > 0). While missing, the pet is invisible, inactive, and grants no bonuses.

PetObject.GoMissing(rounds int)void

Sends the pet away for the given number of rounds, firing PetLeave(). Pass 0 to return the pet immediately, firing PetReturn().

ArgumentTypeDescription
roundsintRounds to be absent. 0 = return immediately.

PartyObject

Groups

Operates on a collection of actors simultaneously. Obtained via ActorObject.GetParty(), GetPartyPresent(), or GetPartyMissing(). All methods apply to every member in the collection.

PartyObject.GetMembers()[]ActorObject

Returns all ActorObjects in the party collection.

PartyObject.SendText(msg string)void

Sends a message to all party members.

PartyObject.GiveQuest(questId string)void

Grants a quest or quest step to all party members.

PartyObject.AddGold(amt int [, bankAmt int])void

Adjusts gold (and optionally bank gold) for all party members.

PartyObject.AddHealth(amt int)void

Adjusts health for all party members.

PartyObject.SetHealth(amt int)void

Sets health to a specific amount for all party members.

PartyObject.AddMana(amt int)void

Adjusts mana for all party members.

PartyObject.Command(cmd string [, waitSeconds float])void

Forces all party members to execute a command.

PartyObject.MoveRoom(destRoomId int)void

Quietly moves all party members to the destination room.

PartyObject.SetResetRoomId(roomId int)void

Sets the logout room for all user party members.

PartyObject.TrainSkill(skillName string, skillLevel int)void

Sets a skill level for all party members (only if greater than current).

PartyObject.GiveBuff(buffId int, source string)void

Applies a buff to all party members.

PartyObject.RemoveBuff(buffId int)void

Removes a buff from all party members without triggering onEnd.

PartyObject.CancelBuffWithFlag(buffFlag string)void

Cancels any buffs with the given flag for all party members.

PartyObject.ChangeAlignment(alignmentChange int)void

Adjusts alignment for all party members.

PartyObject.LearnSpell(spellId string)void

Adds a spell to all party members' spellbooks.

Global

Creates a blank panel layout entirely in script. All opts properties are optional.

OptionDefaultDescription
border"full""full" — every row has side borders. "open" — only first and last rows.
charset"single""single" ┌─┐, "double" ╔═╗, "rounded" ╭─╮, or 8-char literal.

Adds a new vertical slot (column) to the layout. Slots are rendered side by side. Returns a SlotObject for adding rows.

Overrides the border character set for this panel only. Returns panel for chaining.

Sets the minimum inner content width in characters. Returns panel for chaining.

Sets how many label+value pairs per rendered line. 1 (default) or 2. Returns panel for chaining.

Appends an empty spacer row for visual separation. Returns panel for chaining.

Full example — two-panel status display 
function onCommand_status(rest, user, room) {
    var layout = PanelLayoutNew({ border: 'full', charset: 'single', gap: 1, margin: 1 });

    var slot = layout.AddSlot();
    slot.AddRow(['info', 'stats']);

    layout.Panel('info')
        .SetTitle(' .:Character ')
        .SetMinWidth(24)
        .Add('Name:  ', 'N:', user.GetCharacterName(false))
        .Add('Level: ', 'Lvl:', String(user.GetLevel()))
        .Add('Health:', 'HP:', user.GetHealth() + '/' + user.GetHealthMax())
        .Add('Mana:  ', 'MP:', user.GetMana() + '/' + user.GetManaMax());

    layout.Panel('stats')
        .SetTitle(' .:Attributes ')
        .SetMinWidth(28)
        .SetColumns(2)
        .Add('Strength: ', 'Str:', String(user.GetStat('strength')))
        .Add('Vitality: ', 'Vit:', String(user.GetStat('vitality')))
        .Add('Speed:    ', 'Spd:', String(user.GetStat('speed')))
        .Add('Mysticism:', 'Mys:', String(user.GetStat('mysticism')));

    user.SendText(layout.Render());
    return true;
}
