GameTime API Reference

REST endpoints for listing, creating, updating, and deleting calendar configurations. View / Edit

GET /admin/api/v1/gametime Return all calendar configurations gametime.read

Returns a map of all loaded calendar configurations, keyed by calendar name.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/gametime
200 Success
{ "success": true, "data": { "default": { "rounds_per_day": 900, "night_hours": 8, "dusk_hours": 3, "days_per_year": 365, "days_per_week": 7, "sun_count": 1, "moon_count": 1, "months": ["Arvalon", "Beldris", "..."], "zodiac": ["Aardvark", "Albatross", "..."] } } }
GET /admin/api/v1/gametime/{calendar} Return a single calendar configuration gametime.read

Returns the configuration for the named calendar.

ParameterInDescription
calendar requiredpathCalendar name (e.g. default).
curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/gametime/default
200 Success
{"success":true,"data":{"rounds_per_day":900,"night_hours":8,...}}
404 Not Found
{"success":false,"error":"calendar not found: myworld"}
POST /admin/api/v1/gametime Create a new calendar gametime.write

Creates a new named calendar. Returns 400 if the name already exists or validation fails.

FieldTypeDescription
name requiredstringUnique calendar identifier (e.g. myworld).
config.rounds_per_day requiredintGame rounds per day. Minimum 24.
config.night_hoursintHours of night per day (0–23). Default 0.
config.dusk_hoursintHours before night start when dusk colour begins.
config.days_per_year requiredintDays per year. Minimum 1.
config.days_per_week requiredintDays per week. Minimum 1.
config.sun_countintNumber of suns (1–2). Default 1.
config.moon_countintNumber of moons (0–3). Default 0.
config.months required[]stringOrdered list of month names. Must not be empty.
config.zodiac required[]stringList of zodiac animal names. Must not be empty.
curl -s -u admin:password -X POST \ -H "Content-Type: application/json" \ -d '{ "name": "myworld", "config": { "rounds_per_day": 720, "night_hours": 6, "dusk_hours": 2, "days_per_year": 300, "days_per_week": 5, "sun_count": 2, "moon_count": 0, "months": ["Firstmonth", "Secondmonth", "Thirdmonth"], "zodiac": ["Dragon", "Phoenix", "Kraken"] } }' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/gametime
201 Created
{"success":true,"data":{"rounds_per_day":720,"night_hours":6,...}}
400 Validation error or duplicate name
{"success":false,"error":"calendar already exists: myworld"}
PATCH /admin/api/v1/gametime/{calendar} Update an existing calendar gametime.write

Updates the named calendar. The request body is a CalendarConfig object; fields provided overwrite the existing values. The calendar name is always taken from the URL.

ParameterInDescription
calendar requiredpathCalendar name to update.
curl -s -u admin:password -X PATCH \ -H "Content-Type: application/json" \ -d '{ "rounds_per_day": 1200, "night_hours": 10, "dusk_hours": 3, "days_per_year": 365, "days_per_week": 7, "sun_count": 1, "moon_count": 1, "months": ["Arvalon", "Beldris", "Celmara", "Durelin", "Esmira", "Ferulan", "Glimar", "Hestara", "Irinel", "Jorenth", "Keldris", "Luneth"], "zodiac": ["Dragon", "Phoenix"] }' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/gametime/default
200 Success
{"success":true,"data":{"rounds_per_day":1200,"night_hours":10,...}}
404 Not Found
{"success":false,"error":"calendar not found: myworld"}
400 Validation error
{"success":false,"error":"rounds_per_day must be at least 24"}
DELETE /admin/api/v1/gametime/{calendar} Delete a calendar gametime.write

Removes the named calendar from memory and persists the change to gametime.yaml. The default calendar cannot be deleted.

ParameterInDescription
calendar requiredpathCalendar name. Cannot be default.
curl -s -u admin:password -X DELETE \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/gametime/myworld
200 Success
{"success":true}
400 Cannot delete default
{"success":false,"error":"the default calendar cannot be deleted"}
400 Not Found
{"success":false,"error":"calendar \"myworld\" not found"}
