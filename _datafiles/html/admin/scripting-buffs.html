Buff Scripts

Run custom logic when a status effect starts, triggers, or expires.

Script Path

Buff scripts live alongside the buff YAML definition:

Buff definition _datafiles/world/default/buffs/1-illumination.yaml
Buff script _datafiles/world/default/buffs/1-illumination.js

Event Hooks

onStart(actor ActorObject, triggersLeft int)

Called once when the buff is first applied to an actor.

ArgumentTypeDescription
actorActorObjectThe actor receiving the buff.
triggersLeftintNumber of remaining triggers until expiry.
function onStart(actor, triggersLeft) {
    actor.SendText('A warm light surrounds you.');
}
onTrigger(actor ActorObject, triggersLeft int)

Called every time the buff triggers (according to its TriggerRate in the buff definition).

ArgumentTypeDescription
actorActorObjectThe actor with the buff.
triggersLeftintRemaining triggers until expiry.
function onTrigger(actor, triggersLeft) {
    actor.AddHealth(5); // heal 5 HP each trigger
    if (triggersLeft === 1) {
        actor.SendText('The healing light is fading...');
    }
}
onEnd(actor ActorObject, triggersLeft int)

Called when the buff has run its course, right before it is removed from the actor.

ArgumentTypeDescription
actorActorObjectThe actor losing the buff.
triggersLeftintWill be 0 when buff expires naturally.
function onEnd(actor, triggersLeft) {
    actor.SendText('The light fades and darkness returns.');
}
