Test changes on a development server before applying them to a live world.

The three lines represent example racial base intensities — a race with base 1 in a stat grows mostly from natural gains; base 5 pulls further ahead as racial scaling compounds. Solid lines = ValueAdj (effective compressed value). Dashed = raw Value before cap compression. When “Exempt Training & Mods” is on, solid shows the compressed racial floor — training points land on top of it uncapped.
