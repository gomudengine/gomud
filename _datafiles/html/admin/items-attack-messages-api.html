Attack Messages API Reference

REST endpoints for viewing and editing weapon combat messages.

GET /admin/api/v1/items/attack-messages Return all weapon attack message groups

Returns every weapon subtype's attack messages, keyed by subtype. Each subtype contains intensities (prepare, wait, miss, weak, normal, heavy, critical), each with Together and Separate proximity groups containing messages by target audience.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/items/attack-messages
200 Success
{"success":true,"data":{"claws":{"OptionId":"claws","Options":{"prepare":{"Together":{"ToAttacker":["..."],"ToDefender":["..."],"ToRoom":["..."]},"Separate":{...}},...}},...}}
PUT /admin/api/v1/items/attack-messages/{subtype}/{intensity}/{proximity}/{target} Add a new attack message

Appends a new message to the specified subtype, intensity, proximity, and target. The message is saved to the YAML file immediately.

Path parameters:

subtype - weapon subtype (e.g. claws, slashing, generic).
intensity - one of: prepare, wait, miss, weak, normal, heavy, critical.
proximity - together (same room) or separate (different rooms).
target - message audience. Together: toattacker, todefender, toroom. Separate: toattacker, todefender, toattackerroom, todefenderroom.

curl -s -u admin:password \ -X PUT \ -H "Content-Type: application/json" \ -d '{"message":"You lunge at <ansi fg=\"{targettype}\">{target}</ansi> with your <ansi fg=\"item\">{itemname}</ansi>!"}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/items/attack-messages/claws/normal/together/toattacker
200 Success
{"success":true}
400 Bad Request
{"success":false,"error":"unknown subtype: invalid"}
DELETE /admin/api/v1/items/attack-messages/{subtype}/{intensity}/{proximity}/{target}/{index} Delete an attack message by index

Removes the message at the given zero-based index from the specified location. The change is saved to the YAML file immediately.

Path parameters: Same as PUT, plus index - zero-based position of the message to delete.

curl -s -u admin:password \ -X DELETE \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/items/attack-messages/claws/normal/together/toattacker/2
200 Success
{"success":true}
400 Bad Request - invalid index
{"success":false,"error":"index 99 out of range (length 3)"}
400 Bad Request - unknown subtype
{"success":false,"error":"unknown subtype: invalid"}
