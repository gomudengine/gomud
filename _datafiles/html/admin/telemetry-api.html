Telemetry API Reference

REST endpoints for querying and clearing aggregated game event counters.

GET /admin/api/v1/telemetry Query telemetry records

Returns an array of Record objects matching the supplied filters. All parameters are optional; omitting a parameter matches any value for that field. Results are sorted by count descending by default.

ParameterTypeDescription
categorystring optionalOne of item_drop, item_pickup, mob_kill, player_death, item_purchase
itemIdint optionalFilter by item spec ID
mobIdint optionalFilter by mob spec ID
roomIdint optionalFilter by room ID
zonestring optionalExact zone name match
datestring optionalExact date YYYYMMDD. Shorthand for dateFrom + dateTo set to the same value.
dateFromstring optionalStart of date range YYYYMMDD, inclusive
dateTostring optionalEnd of date range YYYYMMDD, inclusive
sortstring optionalasc or desc (default desc). Sorts by count.
curl -s -u admin:password \ "http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/telemetry?category=item_drop&sort=desc"
curl -s -u admin:password \ "http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/telemetry?category=mob_kill&zone=frostfang&dateFrom=20260101&dateTo=20261231"
curl -s -u admin:password \ "http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/telemetry?category=item_drop&itemId=42"
200 Success
{"success":true,"data":[ {"date":"20260510","category":"item_drop","itemid":42,"mobid":7,"roomid":100,"zone":"frostfang","count":14}, {"date":"20260510","category":"item_drop","itemid":42,"mobid":12,"roomid":105,"zone":"frostfang","count":3} ]}
DELETE /admin/api/v1/telemetry Clear matching telemetry records

Removes all records matching the supplied filters and saves the result to disk. Omit all parameters to clear every record. Accepts the same filter parameters as GET except sort.

ParameterTypeDescription
categorystring optionalRestrict deletion to one category
zonestring optionalRestrict deletion to one zone
datestring optionalDelete records for a single day YYYYMMDD
dateFromstring optionalStart of date range to delete, inclusive
dateTostring optionalEnd of date range to delete, inclusive
curl -s -u admin:password \ -X DELETE \ "http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/telemetry?category=item_drop&date=20260101"
curl -s -u admin:password \ -X DELETE \ "http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/telemetry?dateFrom=20260101&dateTo=20260131"
200 Success
{"success":true}
500 Save Error
{"success":false,"error":"telemetry.Save write: ..."}
