Mutators API Reference

REST endpoints for browsing, creating, editing, and deleting room mutator definitions.

GET /admin/api/v1/mutators Return all mutator specs

Returns an array of all loaded MutatorSpec objects.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/mutators
200 Success
{"success":true,"data":[{"MutatorId":"desert-sun","DecayRate":"sunset","RespawnRate":"noon","PlayerBuffIds":[33],...}]}
POST /admin/api/v1/mutators Create a new mutator

Creates a new mutator spec. MutatorId is required and must be unique, lowercase alphanumeric with hyphens/underscores. Buff IDs are validated against loaded buff specs.

curl -s -u admin:password \ -X POST \ -H "Content-Type: application/json" \ -d '{"MutatorId":"my-mutator","PlayerBuffIds":[24],"LightMod":-1}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/mutators
201 Created
{"success":true,"data":{"MutatorId":"my-mutator","PlayerBuffIds":[24],"LightMod":-1}}
400 Bad Request
{"success":false,"error":"mutator already exists: my-mutator"}
GET /admin/api/v1/mutators/{mutatorId} Get a single mutator spec

{mutatorId} is the string mutator ID (e.g. desert-sun).

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/mutators/desert-sun
200 Success
{"success":true,"data":{"MutatorId":"desert-sun","DecayRate":"sunset","RespawnRate":"noon","PlayerBuffIds":[33]}}
404 Not Found
{"success":false,"error":"mutator not found: nonexistent"}
PATCH /admin/api/v1/mutators/{mutatorId} Update mutator properties

Merges the supplied fields into the existing spec. The MutatorId in the body is ignored — the path parameter is authoritative. Buff IDs and LightMod range are validated before saving.

curl -s -u admin:password \ -X PATCH \ -H "Content-Type: application/json" \ -d '{"LightMod":1,"PlayerBuffIds":[24,33]}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/mutators/desert-sun
200 Success
{"success":true,"data":{"MutatorId":"desert-sun","LightMod":1,"PlayerBuffIds":[24,33],...}}
400 Bad Request
{"success":false,"error":"malformed request body: ..."}
404 Not Found
{"success":false,"error":"mutator not found: nonexistent"}
DELETE /admin/api/v1/mutators/{mutatorId} Delete a mutator

Permanently removes the mutator YAML from disk and from the in-memory cache.

curl -s -u admin:password \ -X DELETE \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/mutators/desert-sun
200 Success
{"success":true}
500 Server Error
{"success":false,"error":"mutator not found: nonexistent"}
