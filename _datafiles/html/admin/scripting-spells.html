Spell Scripts

Control the casting, waiting, and resolution phases of spells.

Script Path

Spell scripts live alongside the spell YAML definition:

Spell definition _datafiles/world/default/spells/heal.yaml
Spell script _datafiles/world/default/spells/heal.js

Spell Types & the ARG Parameter

The second argument in all three hooks varies based on the spell's type defined in its YAML:

neutral
ARG is a string — whatever the player typed after cast <spellname>.
harmsingle
ARG is a single ActorObject — the targeted enemy.
helpsingle
ARG is a single ActorObject — the targeted ally.
harmmulti
ARG is an array of ActorObjects — all targeted enemies.
helpmulti
ARG is an array of ActorObjects — all targeted allies.

Event Hooks

onCast(sourceActor ActorObject, ARG varies)

Called when a player attempts to cast the spell. Return false to abort the cast entirely.

Return value: false = abort cast. Any other value = proceed.
ArgumentTypeDescription
sourceActorActorObjectThe player casting.
ARGvariesDepends on spell type — see table above.
function onCast(sourceActor, ARG) {
    if (sourceActor.GetMana() < 10) {
        sourceActor.SendText('You don\'t have enough mana!');
        return false; // abort
    }
}
onWait(sourceActor ActorObject, ARG varies)

Called each round while the player waits for the spell's cast time to complete (for spells with a cast delay).

ArgumentTypeDescription
sourceActorActorObjectThe casting player.
ARGvariesDepends on spell type — see table above.
function onWait(sourceActor, ARG) {
    sourceActor.SendText('You continue channeling the spell...');
}
onMagic(sourceActor ActorObject, ARG varies)

Called when the spell is fully cast and takes effect. For harmful spells, return true to suppress automatic mob retaliation.

Return value: For harm spells — true = suppress mob auto-retaliation. For others — no effect.
ArgumentTypeDescription
sourceActorActorObjectThe caster.
ARGvariesDepends on spell type — see table above.
// helpsingle spell — ARG is the target actor
function onMagic(sourceActor, ARG) {
    var healed = ARG.AddHealth(50);
    ARG.SendText('You feel a wave of healing wash over you (+' + healed + ' HP).');
    sourceActor.SendText('You heal ' + ARG.GetCharacterName(false) + ' for ' + healed + ' HP.');
}
