Keywords API Reference

REST endpoints for viewing and updating keywords: help topics, aliases, direction shortcuts, and map legend overrides.

GET /admin/api/v1/keywords Return all keywords data

Returns the full keywords configuration containing help topics, help aliases, command aliases, direction aliases, and map legend overrides.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/keywords
200 Success
{"success":true,"data":{"Help":{"command":{"configuration":["alias","macros","set"],...},"skill":{"all":["aid","backstab",...]},"admin":{"all":["buff","build",...]}},"HelpAliases":{"brawling":["tackle","brawl"],...},"CommandAliases":{"say":["."],"status":["sta","stat"],...},"DirectionAliases":{"n":"north","s":"south",...},"MapLegendOverrides":{"*":{"$":"Shop","★":"Bank"},"Frostfang":{"!":"Throne Room"}}}}
PATCH /admin/api/v1/keywords Update all keywords data

Replaces the entire keywords configuration. The request body must contain the full keywords structure with all five sections: Help, HelpAliases, CommandAliases, DirectionAliases, and MapLegendOverrides. The file is saved to disk and reloaded in memory.

curl -s -u admin:password -X PATCH \ -H "Content-Type: application/json" \ -d '{"Help":{"command":{"general":["quit","online"]}},"HelpAliases":{"brawling":["tackle"]},"CommandAliases":{"say":["."]},"DirectionAliases":{"n":"north"},"MapLegendOverrides":{"*":{"$":"Shop"}}}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/keywords
200 Success
{"success":true,"data":{"Help":{...},"HelpAliases":{...},"CommandAliases":{...},"DirectionAliases":{...},"MapLegendOverrides":{...}}}
400 Bad Request
{"success":false,"error":"malformed request body: unexpected EOF"}
500 Internal Server Error
{"success":false,"error":"writing keywords file: permission denied"}
