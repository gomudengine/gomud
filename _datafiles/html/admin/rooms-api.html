Rooms & Zones API Reference

REST endpoints for browsing, creating, editing, and deleting rooms and zone configurations.

Tags
GET /admin/api/v1/tags List all registered room tags

Returns every room tag registered by loaded modules, each with its module name. Tags are sorted by module then by registration order.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/tags
200 Success
{"success":true,"data":[{"tag":"storage","module":"storage"},{"tag":"no-combat","module":"zombiemode"}]}
Zones
GET /admin/api/v1/zones List all zones with room counts

Returns an array of zone summaries sorted by name.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/zones
POST /admin/api/v1/zones Create a new zone

Creates a new zone with a root room. Name must be at least 2 characters (letters, numbers, spaces, underscores).

curl -s -u admin:password \ -X POST -H "Content-Type: application/json" \ -d '{"Name":"The Arctic"}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/zones
PATCH /admin/api/v1/zones/{zonename} Update zone configuration

Merges supplied fields into the zone config. Name and RoomId cannot be changed.

curl -s -u admin:password \ -X PATCH -H "Content-Type: application/json" \ -d '{"DefaultBiome":"cave","MobAutoScale":{"Minimum":5,"Maximum":15}}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/zones/Frostfang
DELETE /admin/api/v1/zones/{zonename} Delete a zone and all its rooms

Permanently deletes the zone, its configuration, and all rooms in the zone. Fails if any room in the zone has players present.

curl -s -u admin:password \ -X DELETE \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/zones/The%20Arctic
Rooms
GET /admin/api/v1/rooms List rooms (paginated)

Returns a paginated list of room summaries. Query params: zone (filter by zone), search (filter by ID or title), page (default 1), perPage (default 50, max 200).

curl -s -u admin:password \ "http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/rooms?zone=Frostfang&page=1&perPage=50"
200 Success
{"success":true,"data":{"rooms":[{"RoomId":1,"Zone":"Frostfang","Title":"Town Square","Biome":"city","ExitCount":4}],"total":150,"page":1,"perPage":50}}
GET /admin/api/v1/rooms/biomes List all biome definitions

Returns all loaded biome definitions for use in dropdowns.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/rooms/biomes
POST /admin/api/v1/rooms Create a new room

Creates a new room in the specified zone with an auto-assigned ID and default title/description.

curl -s -u admin:password \ -X POST -H "Content-Type: application/json" \ -d '{"Zone":"Frostfang"}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/rooms
GET /admin/api/v1/rooms/{roomId} Get full room template

Returns the complete room template data for editing. {roomId} must be a numeric ID.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/rooms/1
PATCH /admin/api/v1/rooms/{roomId} Update room properties

Merges supplied fields into the room template. RoomId in the body is ignored. Title and Description are required.

curl -s -u admin:password \ -X PATCH -H "Content-Type: application/json" \ -d '{"Title":"Grand Hall","Description":"A magnificent hall."}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/rooms/1
GET /admin/api/v1/rooms/{roomId}/script Get room script

Returns the JavaScript source for the room's script file. script is an empty string if no script file exists.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/rooms/1/script
200 Success
{"success":true,"data":{"script":"function onEnter(actor) { ... }"}}
PUT /admin/api/v1/rooms/{roomId}/script Replace or delete room script

Writes the supplied JavaScript to the room's .js script file. Send an empty script string to delete the script file.

curl -s -u admin:password \ -X PUT -H "Content-Type: application/json" \ -d '{{`{"script":"function onEnter(actor) { actor.SendText(\"Welcome!\"); }"}`}}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/rooms/1/script
DELETE /admin/api/v1/rooms/{roomId} Delete a room

Permanently removes a room's template and instance files. Cannot delete a zone's root room or a room with players present.

curl -s -u admin:password \ -X DELETE \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/rooms/42
