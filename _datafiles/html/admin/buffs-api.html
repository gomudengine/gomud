Buffs API Reference

REST endpoints for browsing, editing, and deleting buff (status effect) definitions.

GET /admin/api/v1/buffs Return all buff specs and flag specs buffs.read

Returns specs (buffId → BuffSpec) and flags (an array of FlagSpec objects).

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/buffs
200 Success
{"success":true,"data":{"flags":[{"Flag":"hidden","Name":"Hidden","Description":"Marks the character as hidden or stealthed, concealing them from others.","Locked":true}],"specs":{"1":{"BuffId":1,"Name":"Fast Healing","TriggerRate":"1 round","TriggerCount":10,"Flags":["hidden"]}}}}
GET /admin/api/v1/buffs/{buffId} Get a single buff spec buffs.read

{buffId} must be a numeric buff ID.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/buffs/1
200 Success
{"success":true,"data":{"BuffId":1,"Name":"Fast Healing","TriggerRate":"1 round","TriggerCount":10,"Flags":["hidden"]}}
404 Not Found
{"success":false,"error":"buff not found: 999"}
PATCH /admin/api/v1/buffs/{buffId} Update buff properties buffs.write

Merges the supplied fields into the existing spec. The BuffId in the body is ignored - the path parameter is authoritative. The buff is re-validated (including TriggerRate → RoundInterval conversion) before saving.

curl -s -u admin:password \ -X PATCH \ -H "Content-Type: application/json" \ -d '{"TriggerCount":20,"Flags":["poison","hidden"]}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/buffs/1
200 Success
{"success":true,"data":{"BuffId":1,"Name":"Fast Healing","TriggerRate":"1 round","TriggerCount":20,"Flags":["poison","hidden"]}}
400 Bad Request
{"success":false,"error":"malformed request body: ..."}
404 Not Found
{"success":false,"error":"buff not found: 999"}
DELETE /admin/api/v1/buffs/{buffId} Delete a buff and its script buffs.write

Permanently removes the buff YAML and associated JS script from disk and from the in-memory cache.

curl -s -u admin:password \ -X DELETE \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/buffs/1
200 Success
{"success":true}
404 Not Found
{"success":false,"error":"buff not found: 999"}
GET /admin/api/v1/buffs/{buffId}/script Get the buff's JavaScript buffs.read

Returns the raw script content. script is an empty string if no script file exists.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/buffs/1/script
200 Success
{"success":true,"data":{"script":"function onStart(actor) {\n}\n"}}
404 Not Found
{"success":false,"error":"buff not found: 999"}
PUT /admin/api/v1/buffs/{buffId}/script Replace (or delete) the buff's JavaScript buffs.write

Writes the provided script string to the buff's .js file. Send an empty string to delete the script file.

curl -s -u admin:password \ -X PUT \ -H "Content-Type: application/json" \ -d '{"script":"function onStart(actor){}"}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/buffs/1/script
200 Success
{"success":true}
400 Bad Request
{"success":false,"error":"malformed request body: ..."}
404 Not Found
{"success":false,"error":"buff not found: 999"}

Buff Flags API Reference

REST endpoints for browsing, editing, and deleting buff flag definitions. Locked flags cannot be edited or deleted.

GET /admin/api/v1/buff-flags Return all flag specs (sorted) buffs.read

Returns an array of FlagSpec objects sorted by flag identifier.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/buff-flags
200 Success
{"success":true,"data":[{"Flag":"perma-gear","Name":"Unremovable Gear","Description":"Prevents equipped items from being removed while the buff is active.","Locked":true}]}
GET /admin/api/v1/buff-flags/{flag} Get a single flag spec buffs.read

{flag} is the flag identifier, e.g. perma-gear.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/buff-flags/perma-gear
200 Success
{"success":true,"data":{"Flag":"perma-gear","Name":"Unremovable Gear","Description":"Prevents equipped items from being removed while the buff is active.","Locked":true}}
404 Not Found
{"success":false,"error":"flag not found: nope"}
POST /admin/api/v1/buff-flags Create a new flag buffs.write

Creates a new flag definition. New flags are always created unlocked; the Locked field in the body is ignored. The Flag identifier must be lowercase alphanumeric with hyphens/underscores, starting with a letter.

curl -s -u admin:password \ -X POST \ -H "Content-Type: application/json" \ -d '{"Flag":"my-flag","Name":"My Flag","Description":"Does something."}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/buff-flags
201 Created
{"success":true,"data":{"Flag":"my-flag","Name":"My Flag","Description":"Does something.","Locked":false}}
400 Bad Request
{"success":false,"error":"flag already exists: my-flag"}
PATCH /admin/api/v1/buff-flags/{flag} Update a flag's name/description buffs.write

Updates the Name and/or Description. The flag identifier cannot be changed. Locked flags reject this request with a 400.

curl -s -u admin:password \ -X PATCH \ -H "Content-Type: application/json" \ -d '{"Name":"My Renamed Flag"}' \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/buff-flags/my-flag
200 Success
{"success":true,"data":{"Flag":"my-flag","Name":"My Renamed Flag","Description":"Does something.","Locked":false}}
400 Bad Request (locked)
{"success":false,"error":"flag \"perma-gear\" is locked and cannot be edited"}
404 Not Found
{"success":false,"error":"flag not found: nope"}
DELETE /admin/api/v1/buff-flags/{flag} Delete a flag buffs.write

Permanently removes the flag YAML from disk and the in-memory cache. Locked flags reject this request with a 400.

curl -s -u admin:password \ -X DELETE \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/buff-flags/my-flag
200 Success
{"success":true}
400 Bad Request (locked)
{"success":false,"error":"flag \"perma-gear\" is locked and cannot be deleted"}
GET /admin/api/v1/buff-flags/{flag}/yaml View the flag's raw YAML buffs.read

Returns the raw on-disk YAML for the flag along with its relative path.

curl -s -u admin:password \ http://{{.CONFIG.FilePaths.WebDomain}}/admin/api/v1/buff-flags/perma-gear/yaml
200 Success
{"success":true,"data":{"yaml":"flag: perma-gear\nname: ...","path":"buffs-flags/perma-gear.yaml"}}
404 Not Found
{"success":false,"error":"flag not found: nope"}
